AjouMu
======

설치방법

설치할 프로그램

rails 버전 3: http://rubyonrails.org/download

git: http://www.git-scm.com/

`git clone https://github.com/co2meal/ajoumu.git`

`cd ajoumu`

`bundle install`

`rake db:migrate`

`rails server`


하면 됩니다.
브라우저에서 http://localhost:3000하면 끝.
